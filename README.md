# MERN-Todo
Установите зависимости
* `npm install`
* `npm run installFront`

Запуск одной командой 
* `npm start`
  
Используемые технологии 
* Node
* Express
* Mongoose
* MongoDB
* JWT
* React
* React-hook-form
* React-Router
* React-Redux
* Redux Toolkit
* RTK Query
* react-toastify
* axios
* bootstrap
* sass
* vite
